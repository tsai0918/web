咖啡

咖啡（英語：coffee）是采經過烘焙過程的咖啡豆（咖啡屬植物的種子）所製作沖泡的飲料。咖啡是人類社會流行範圍最為廣泛的飲料之一，也是重要經濟作物，其為全球期貨貿易額度第二高（最高為石油）。

主要成分

  • 蔗糖：烘焙過程中部分會熱裂解為甲酸、醋酸、乳酸、甘醇酸，故含糖量愈高的生豆
  • 咖啡因：咖啡的主要活性成分。略帶苦味，熔點高達237℃，因此烘焙好的咖啡熟豆，其咖啡因幾乎完整保留下來，並在萃取時會融入杯中。
  • 綠原酸：淺焙至中焙（一爆至二爆）中，約有50%的綠原酸會降解為奎寧酸（酸澀），而咖啡亦是人類攝取綠原酸的主要來源。
